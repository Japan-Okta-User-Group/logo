## 禁止事項
* ロゴデータを変形（長体・平体・斜体）しないでください
* 陰影をつけて表示しないでください
* グラデーション背景で表示しないでください
* 周辺にその他の図形を表示しないでください
* 識別を損なう表示をしないでください
* ふちどりで表示しないでください
* 煩雑な背景（パターン・写真等）に配置しないでください
* 本ロゴデータの販売、再分布をしないでください
